User Access Review is a private Chrome Extension that I created as a contactor for Fresh Consulting to parse SaaS websites and track which employees are enrolled in what services to reduce company spend on unnecessary subscriptions.

More info
  • Securely stored data inside Firebase with UI inside the extension to efficiently manage the data
  • Implemented seamless OAuth 2.0 authentication for Google and other providers
  • Integrated the extension with Google Workspace using the Google Directory API
  • Worked in a CI-CD environment with Bitbucket Pipelines using linting via ESLint and testing via Jasmine
